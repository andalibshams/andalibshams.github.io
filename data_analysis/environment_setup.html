Data Analysis Environment setup

  1. Download Python- install miniconda. It is easy to organize and create clear environment using Miniconda (also does not require admin right to install).

  2. Follow start menu → Anaconda 3 64-bit → Anaconda prompt.

  3. Create an environment in the anaconda prompt by command. Here myenv is the environment name I have used.

    conda create --name myenv

    Creating environment ensures that in future if we install different version of python, the current environment will not conflict with the future python version. So, we don’t need to bother with compatibility issues.

  4. Now we need activate ‘myenv’ by command-

    conda activate myenv

  5. Install numpy package into myenv- (numpy package is helpful to handle n-dimensional array)

    conda install numpy

  6. Install panda package into myenv- (panda is a data analysis and manipulation tool. )

    conda install -c anaconda pandas

  7. Install matplotlib package into myenv- (Matplotlib is a data visualization library for python )

    conda install -c conda-forge matplotlib

  8. I personally prefer jupyter lab instead of the notebook version. In jupyter lab, we can output images in new window, and simply holding alt+click takes us to function definition. I like these features.

    conda install -c conda-forge jupyterlab
